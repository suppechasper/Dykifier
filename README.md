# Word Correlation Explorer
Shiny app for hallucinating dykes.


## Installation ##

1. Install R
2. Start R
3. In R run the commands:
    1. install.packages( c("shiny", "RColorBrewer", "devtools") )
    2. library("devtools")
    3. devtools::install_github( "suppechasper/Dykifier") )
     
    
## Running ##

1. Start R.
2. In R run the commands:
   1. library(Dykifier)
   2. dykify()
